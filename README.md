## Français

### Présentation
AutoCloseDoors est un script Space Engineers qui referme automatiquement les portes. Il surveille les groupes de portes AUTOCLOSEDOOR et applique un delai configurable. Le projet vise une automatisation simple et fiable pour bases et vaisseaux.

### Fonctionnalités
- Fermeture automatique des portes surveillees.
- Groupes AUTOCLOSEDOOR simples ou nommes.
- Commande scan pour rafraichir la detection.
- Limitation volontaire a la grille du Programmable Block.

### Installation
Copie Program.cs dans un Programmable Block Space Engineers, puis cree un ou plusieurs groupes AUTOCLOSEDOOR.

### Utilisation
Lance le Programmable Block et laisse le script surveiller les portes du groupe. Execute scan apres l'ajout, la suppression ou le renommage de portes.

## English

### Overview
AutoCloseDoors is a Space Engineers script that automatically closes doors. It watches AUTOCLOSEDOOR door groups and applies a configurable delay. The project provides simple and reliable automation for bases and ships.

### Features
- Automatic closing for watched doors.
- Simple or named AUTOCLOSEDOOR groups.
- scan command to refresh detection.
- Intentionally limited to the Programmable Block grid.

### Installation
Copy Program.cs into a Space Engineers Programmable Block, then create one or more AUTOCLOSEDOOR groups.

### Usage
Run the Programmable Block and let the script watch the grouped doors. Execute scan after adding, removing or renaming doors.
